# Think

`think` is a small skill repo for AI coding tools. It provides a `/think` workflow
for turning messy thoughts into a structured breakdown with threads, open
questions, and action items.

## Repo

Published at: `fheinonen/think`

## Install

```bash
npx skills add fheinonen/think
```

## What It Does

- Captures a raw thought dump in one pass
- Organizes it into clearer threads and next steps
- Helps continue, refine, and close out thinking sessions
- Saves sessions as portable markdown files

## Usage

Invoke `/think`, or ask your AI tool to help you think through a problem.

Typical flow:

1. Dump everything on your mind.
2. Let the tool structure the problem.
3. Refine threads, questions, or action items.
4. Save the session for later.

## Session Management

- Start: "I need to think through something"
- List: "List my thinking sessions"
- Continue: "Continue the migration session"
- Resolve: "Mark the migration session as done"
